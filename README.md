# Minor_project_Python

## Calculator
A Python command-line calculator that uses *args to perform addition, subtraction, multiplication, and division on any number of user inputs. The project demonstrates functions, loops, user input handling, lists, and modular programming.

## Number Guessing Game
A Python number guessing game that generates a random number using the random module and challenges the user to guess it with a simple hint system. The project demonstrates loops, conditional statements, user input validation, and replay functionality.
